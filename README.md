# Haarun Taofeek Photography Portfolio

## Overview
This is the official photography portfolio of **Haarun Taofeek**, a Nigerian photographer whose work spans **portraits**, **fashion brands**, and **music concert photography**. The website reflects a minimalist and expressive design approach, highlighting visual storytelling through light, emotion, and creativity.
